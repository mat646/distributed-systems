# Distributed Systems Course

Implementation of Token Ring with Inet sokets (UDP, TCP)  
  
## Usage

### TCP

```bash
$ make all  

$ make run-client-1

$ make run-client-2

$ make run-client-3

$ make run-logger

$ make run-logger
``` 

### UDP 

```bash
$ make all  

$ make udp-run-client-1

$ make udp-run-client-2

$ make udp-run-client-3

$ make run-logger

$ make run-logger
```
